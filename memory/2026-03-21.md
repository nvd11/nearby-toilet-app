### Workflow: Web Search to Direct File Download
- **Trigger:** When asked to download reports or files from the web.
- **Method:** 
  1. Use `web_search` to locate the official direct download link (e.g., PDF URL) of the requested document.
  2. Use `exec` to run `wget -q -O <filename> <URL>` to download it directly to the workspace.
  3. Use `exec` to run `ls -lh <filename>` to verify the download size and success.
- **Note:** This bypasses the need for UI browser automation, achieving a fast, headless download.
