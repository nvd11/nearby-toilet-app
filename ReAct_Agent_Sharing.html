ReAct Agent

Reasoning + Acting in LLMs


Presenter: Boss Jason

What is ReAct?


  • 🧠 Reasoning (思考): 像人类一样的思维链 (Chain of Thought)
  • 🛠️ Acting (行动): 调用工具与外部世界交互 (API, Search)
  • ✨ Synergy: 思考指导行动，行动结果反哺思考！

The ReAct Core Loop


1. Thought 🤔

2. Action ⚡

3. Observation 👁️


🔁 Repeat until solved!

Why do we need it?


  • ❌ 传统大模型容易幻觉 (Hallucination)且数据滞后。
  • ✅ ReAct 允许模型读取实时数据 (Real-time data)。
  • ✅ 在出错时能自我修正 (Self-correction)。

A Quick Trace


// 目标：查一下东京的天气
Thought: I need to find the current weather in Tokyo.
Action: weather_api("Tokyo")
Observation: It is currently 22°C and sunny.
Thought: Now I have the info. I can answer.
Final Answer: The weather in Tokyo is 22°C and sunny.
